A countdown for OBS
that you actually control.

Open a control panel in one tab, drop the overlay URL into OBS as a browser source. The two stay in lockstep over a websocket. Style it once, your settings persist.

What it actually does

Make a stopwatch or a count-down

Set a duration and run it down. Or start from zero and let it count. End behaviors: stop, go negative, hide, or fire confetti.

Real-time sync

A websocket keeps the control panel and every overlay in lockstep.

Multi-timer, persisted

Each timer is isolated by URL. Every timer is saved in your browser if you need them later.
