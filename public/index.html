OBS Timer

Free countdown and countup timer overlay for OBS Studio

Quick Start

Enter a room name to create your timer.
Use any name you like, it's just for separating different timers.

Room names can contain letters, numbers, hyphens, and underscores.

How It Works

Count Down

Count down from a set duration. Choose what happens at zero.

Count Up

Count up from zero. Great for tracking elapsed time with no end limit.

Customizable

Adjust font, size, colors, shadows, and more from the control panel.

Real-Time Sync

Control panel and OBS overlay updates instantly via WebSocket.

Persistent

Timer state and settings are saved automatically.

OBS Ready

Transparent background overlay designed for OBS browser sources.

OBS Timer is free and open source. Add a browser source in OBS pointing to your overlay URL.
